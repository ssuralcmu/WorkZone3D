# WorkZone3D

A repository for the multimodal **WorkZone3D** dataset supporting the paper:

**WorkZone3D: A Multimodal Dataset for 3D Work Zone Perception in Autonomous Driving**  
Presented at **IEEE/CVF Winter Conference on Applications of Computer Vision (WACV) 2026**

---

## Paper

- **OpenAccess PDF:**  
https://openaccess.thecvf.com/content/WACV2026/papers/Sural_WorkZone3D_A_Multimodal_Dataset_for_3D_Work_Zone_Perception_in_WACV_2026_paper.pdf
